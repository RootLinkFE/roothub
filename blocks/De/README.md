# Muji quickstart boilerplate

![](https://img.shields.io/badge/typescript-enabled-blue.svg)

## Usage

#### Start a dev process

```bash
yarn start
# or
# npm start
```

#### Build production bundle

```bash
yarn build
# or
# npm run build
```
